Semantic And Non-Semantic Elements:

semantic element in css

A semantic element clearly describes its meaning to both the browser and the developer.

  1. article
  2. aside
  3. details
  4. figcaption
  5. figure
  6. header
  7. footer
  8. main
  9. mark
  10. nav
  11. section
  12. summary
  13. time
non semantic element

Non-semantic elements are similar to semantic elements, but they do not have any meaningful content.

  1. div
  2. span
